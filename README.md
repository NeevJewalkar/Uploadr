## Downloading
Right now, there is no seperate build, so to launch it, first clone the repo

`$ git clone https://github.com/NeevJewalkar/Uploadr.git`

the go to the main file

`$ cd Uploadr`

install electron

`$ npm install electron`
 
and then run your file!
 
`$ npm start`
